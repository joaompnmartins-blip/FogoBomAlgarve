📥 Importar Parcelas

Carregue ficheiros GeoJSON (.geojson, .json) ou KML (.kml) para importar parcelas para o mapa. IMPORTANTE os ficheiros a importar devem estar no sistema de coordenadas WGS84 (EPSG:4326)
Nota: Ao importar, a parcela ficará pronta para desenho. Preencha os dados do formulário (nome, responsável, etc.) e clique em "Guardar" para registar na base de dados.
